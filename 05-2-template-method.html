Template Method

Présentation

Le pattern Template Method permet à des classes enfants de modifier certaines étapes d'un algorithme définit dans une classe parent.

refactoring.guru

Exemple théorique

Dans cet exemple, nous allons utiliser le design pattern Template Method pour gérer un upload de fichier. Pour chaque upload, plusieurs traitements doivent être effectués (déplacer le fichier, le renommer...) mais ces traitements peuvent changer en fonction du type de fichier uploadé (image, vidéo...).
